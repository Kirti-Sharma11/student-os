# AI Student Productivity OS 🚀

A production-level MERN stack platform for students featuring:

- JWT Authentication
- GitHub Developer Profiles
- Placement Tracker
- LeetCode Analytics
- AI Roadmap Generator
- Resume ATS Checker
- Pomodoro Productivity System

## Tech Stack

Frontend:
- React
- TailwindCSS
- Axios
- React Router

Backend:
- Node.js
- Express.js
- MongoDB Atlas
- JWT Authentication

## Features

- Secure Login/Register
- Protected Routes
- Dynamic GitHub API Integration
- Modern Dashboard UI
- Developer Portfolio System
